Scripts and processess to help you create LXC's, VM's and Docker Containers

Goal is to standardize the creation process, so you can spin up a new service with minimal config and time.

Focused around proxmox right now, but most will work on other linux systems utilizing kvm & lxc.

# How was AI used in this repo?
I use AI as a search engine, linter and shellcheck.  AI helps me aggregate information, explains concepts and looks for mistakes.  But I'm ultimately responsible for each block of code, and I will not include a function unless I understand what it does.
AI is not a replacement for me, it is a replacement for Google.  Where just as with Google, I need to validate search results and confirm accuracy before adding them to my project.
